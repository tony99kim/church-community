# 교회 커뮤니티 앱 - 프로젝트 개요

## 프로젝트명
**ChurchHub** (지역 청년 교회 커뮤니티 플랫폼)

## 목적
지역 청년들을 위한 온라인 커뮤니티 공간 제공
- 지역 홍보 및 소식 공유
- 교회 행사 안내
- 청년 간 자유로운 소통 게시판

## 핵심 사용자
| 구분 | 설명 |
|------|------|
| 일반 회원 | 가입 후 게시글/댓글 작성, 행사 참여 |
| 관리자 | 게시판 관리, 회원 관리, 행사/공지 등록 |
| 비회원 | 공개 게시판 조회만 가능 |

---

## 주요 기능

### 사용자 사이트
1. **회원 관리** - 회원가입, 로그인, 프로필 관리, 비밀번호 찾기
2. **커뮤니티 게시판** - 자유게시판, 나눔게시판, 질문게시판 등 다중 카테고리
3. **행사 게시판** - 교회 행사 일정 공지 및 참여 신청
4. **지역 홍보** - 지역 소식, 봉사활동, 모임 홍보
5. **댓글/좋아요** - 게시글에 댓글 및 좋아요 기능
6. **검색** - 게시글 및 회원 검색
7. **알림** - 댓글/좋아요 알림

### 관리자 사이트
1. **대시보드** - 가입자 수, 게시글 수, 방문자 통계
2. **회원 관리** - 회원 목록, 권한 변경, 계정 정지/해제
3. **게시판 관리** - 카테고리 생성/수정/삭제, 게시글 삭제/숨기기
4. **행사 관리** - 행사 등록, 수정, 참여자 목록 조회
5. **신고 관리** - 신고된 게시글/댓글 처리
6. **공지사항** - 전체 공지 등록
