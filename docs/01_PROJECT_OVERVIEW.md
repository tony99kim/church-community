# ChurchHub - 프로젝트 개요

## 프로젝트명
**ChurchHub** (지역 청년 교회 커뮤니티 플랫폼)

## 목적
지역 청년들을 위한 온라인 커뮤니티 공간 제공
- 지역 홍보 및 소식 공유
- 교회 행사 안내
- 청년 간 자유로운 소통 게시판

## 핵심 사용자
| 구분 | 설명 |
|------|------|
| 일반 회원 | 가입 후 게시글/댓글 작성, 좋아요, 신고 |
| 관리자 | 게시판 관리, 회원 관리, 신고 처리 |
| 비회원 | 공개 게시판 조회만 가능 |

---

## 구현 현황

### ✅ 완료된 기능

#### 사용자 사이트
- **회원 관리** - 회원가입, 로그인, 로그아웃, JWT 인증 (Access + Refresh Token)
- **커뮤니티 게시판** - 다중 카테고리, 게시글 CRUD, 페이지네이션, 검색 (키워드+카테고리)
- **댓글** - 댓글/대댓글 CRUD, 댓글 수 카운트 (대댓글 cascade 삭제)
- **좋아요** - 게시글 좋아요/취소
- **신고** - 게시글/댓글 신고
- **알림** - 댓글 알림
- **SEO** - 게시글별 동적 메타태그 (og/twitter)

#### 관리자 사이트 (`/admin`)
- **대시보드** - 가입자 수, 게시글 수 통계
- **회원 관리** - 회원 목록, 권한 변경, 계정 정지/삭제
- **게시판 관리** - 카테고리 생성/수정/삭제, 게시글 삭제
- **신고 관리** - 신고 목록 조회 및 처리

### ❌ 미구현 기능
- 행사 게시판 (행사 등록, 참여 신청)
- 지역 홍보 게시판
- 파일 첨부 (이미지 업로드)
- 이메일 인증 / 비밀번호 찾기
- 비회원 공개 게시판 조회

## 기술 스택

### 백엔드
| 기술 | 버전 | 용도 |
|------|------|------|
| Java | 17 | 기본 언어 |
| Spring Boot | 3.2.5 | 백엔드 프레임워크 |
| Spring Security | 6.x | 인증/인가 |
| Spring Data JPA | - | ORM |
| PostgreSQL | - | 메인 DB (Supabase) |
| Redis | - | 토큰 블랙리스트 (Upstash) |
| JWT | - | 토큰 기반 인증 |
| Gradle | 8.x | 빌드 도구 |

### 프론트엔드
| 기술 | 용도 |
|------|------|
| Next.js 14 (App Router) | SSR/SSG, 사용자+관리자 통합 |
| TypeScript | 타입 안전성 |
| Tailwind CSS | 스타일링 |
| Zustand | 상태 관리 |

### 인프라 / DevOps
| 기술 | 용도 |
|------|------|
| Fly.io | 백엔드 배포 |
| Vercel | 프론트엔드 배포 |
| Supabase | PostgreSQL 호스팅 |
| Upstash | Redis 호스팅 |
| GitHub Actions | 백엔드 자동 배포 (main 브랜치 push 시) |
