# 염리동 청년 커뮤니티 플랫폼 — 피벗 설계 문서

**작성일:** 2026-07-23  
**기존 프로젝트:** ChurchHub (church-community)  
**방향:** 염리동 지역 특화 청년 커뮤니티 플랫폼으로 피벗

---

## 1. 프로젝트 컨셉

**느슨하게 연결된 염리동 청년 커뮤니티.**

- 염리동 12개 교회 청년들을 하나의 플랫폼으로 연결
- 교회 안 사람이든 밖의 사람이든 염리동에 사는/오는 청년 모두 환영
- 안전하고 부담 없는 공간 → 자연스럽게 관계 형성 → 소모임·봉사·전도로 이어짐
- 잘 자리잡으면 이후 지역 확장 (마포·공덕 → 전국) 계획

## 3. 사용자 역할 (4단계)

| Role | 설명 | 기존 매핑 |
|------|------|-----------|
| `SUPER_ADMIN` | 개발자(운영자), 모든 권한 | 기존 SUPER_ADMIN 유지 |
| `CHURCH_MANAGER` | 교회별 담당자, 공간·물품 대여 승인 | 기존 ADMIN → 개명·재활용 |
| `PASTOR` | 교동협의회 목사님, 신앙 Q&A 답변 | 신규 추가 |
| `USER` | 일반 청년 | 기존 USER 유지 |

```
염리동 청년 커뮤니티
├── 홈 (랜딩 — 공개)
├── 처음 오셨나요? / 염리동 웰컴 (공개)
├── 함께하는 교회 (공개)
├── 행사 안내 (공개)
│   ├── 동네 모임
│   ├── 신앙 모임
│   ├── 섬김 모임
│   └── 교회별 행사
├── 웰컴 테이블 (공개 — 행사 하위 or 별도)
├── 커뮤니티 (로그인 필요)
│   ├── 공지 (NOTICE — 운영진만 작성)
│   ├── 자유게시판 (일상·생활 정보·잡담)
│   └── 소모임 모집 (같이 밥·운동·산책 등)

└── 지역 섬김 (공개)
```

## 5. 기능 상세

### 5-1. 염리동 웰컴 (`/welcome`)
- 새로 이사 온 청년·1인 가구·청년주택 입주자 대상
- **웰컴 키트 신청** — 이름·연락처·주소 폼 → 담당자에게 알림
- **동네 생활 가이드** — 마트·병원·카페·도서관·운동시설 등 정보 (정적 콘텐츠 + 관리자 편집)
- **청년 도움 링크** — 청년정책·주거지원·마음건강·복지관·주민센터 (외부 링크 목록)
- **교회 연결 문의** — 부담 없이 문의하는 폼 (이름·연락처·메시지 → 담당자 이메일/알림)

### 5-2. 교회 소개 (`/churches`)
- 교회 목록 (카드형, 공개)
- 각 교회 정보: 이름·위치·주일예배시간·청년모임여부·담당자 연락처·분위기 한 줄 소개·홈페이지/인스타 링크
- SUPER_ADMIN / CHURCH_MANAGER가 관리 (교회별 담당자는 자기 교회만)

### 5-3. 행사 안내 (`/events`) — 기존 Event 도메인 확장
- 기존 EventStatus, EventParticipant 재사용
- **카테고리 추가:** 동네모임 / 신앙모임 / 섬김모임 / 교회별행사 / 웰컴테이블
- 행사 등록: CHURCH_MANAGER 이상
- 참여 신청: 로그인 유저

### 5-4. 웰컴 테이블 (`/welcome-table`)
- 행사 카테고리 중 하나 (별도 페이지 or 행사 목록 필터)
- 월 1회 또는 격월 — 8~15명 소규모 공동 식사
- 참가 신청 폼 (이름·연락처·알레르기 등)
- 핵심 콘텐츠로 홈에서도 강조

### 5-5. 공간 대여 (`/spaces`)
- **공간 목록** — 공개 열람 (교회 주방·소그룹실·카페형·예배실·회의실 등)
- **신청하기** — 로그인 필요, 날짜/시간/인원/목적/책임자 연락처 입력
- **승인 흐름:** 신청 → CHURCH_MANAGER 확인 → 승인/거절 → 신청자 알림
- 자동 예약 없음, 담당자 확인 후 결정

### 5-6. 물품 대여 (`/items`)
- **카테고리:** 이사/정리 / 청소 / 생활 / 행사
- **물품 목록** — 공개 열람 (재고 수량 표시)
- **신청하기** — 로그인 필요, 대여기간·수량·연락처 입력
- **승인 흐름:** 신청 → CHURCH_MANAGER 확인 → 승인/거절 → 신청자 알림
- **약관 동의 필수:** 대여기간·파손 책임·반납 장소·신분 확인·담당자 승인 후 대여

### 5-7. 신앙 질문 / 기도 요청 (`/faith`)
- **탭 구성:** 신앙 질문 / 기도 요청 / 교회 연결 문의
- **신앙 질문:** 익명 가능, PASTOR가 답변 달기, 공개/비공개 선택
- **기도 요청:** 공개/비공개 선택, 성도들이 "함께 기도할게요" 반응
- **교회 연결 문의:** 가까운 교회·청년 모임·예배 시간 안내 요청 폼

### 5-8. 지역 섬김 (`/service`)
- 봉사 기회 목록 (공개)
- 카테고리: 복지관 연계 / 청년주택 환대 / 독거 어르신 / 환경 정화 / 여름성경학교 / 지역 축제 등
- 참여 신청: 로그인 유저
- 기본적으로 Event 도메인 재활용 가능

## 6. 기술 스택 (변경 없음)

| 구분 | 스택 |
|------|------|

## 7. 기존 코드 재사용 / 변경 계획

| 기존 도메인 | 처리 |
|------------|------|
| Auth (JWT, login, register) | 유지, PASTOR role 추가 |
| User entity | 유지, role enum 수정 |
| Post/Comment/Category | 커뮤니티 게시판으로 재활용 — 카테고리 3개로 정리 (공지·자유게시판·소모임모집) |
| Event | 확장 (카테고리 추가, 웰컴테이블·섬김 포함) |
| Notification | 유지, 대여 승인 알림 추가 |
| Report | 유지 (신고 시스템) |
| Admin 대시보드 | 교회·공간·물품·대여 관리 추가 |

### 신규 추가할 도메인
- `Church` — 교회 정보 (이름·위치·예배시간·담당자 등)
- `Space` — 공간 정보 + `SpaceRental` 대여 신청/승인
- `Item` + `ItemCategory` — 물품 정보 + `ItemRental` 대여 신청/승인
- `WelcomeKit` — 웰컴 키트 신청
- `FaithQuestion` — 신앙 질문 (익명 지원, 목사 답변)
- `PrayerRequest` — 기도 요청 (공개/비공개)
